# Spring AI Oracle

Initial scaffold for Spring AI model bindings for Oracle Cloud Infrastructure Generative AI.

This subproject is intentionally separate from the existing Spring Cloud OCI Maven tree and uses `org.springframework.ai:spring-ai-parent:2.0.0-M6` as its parent. Runtime model, auto-configuration, and starter implementations will be added in later changes.

## Modules

| Module | Description |
|--------|-------------|
| `models/spring-ai-oracle` | Model binding module prepared for OCI Generative AI inference dependencies. |
| `auto-configurations/models/spring-ai-autoconfigure-model-oracle` | Spring Boot auto-configuration module placeholder. |
| `starters/spring-ai-starter-model-oracle` | Spring Boot starter module placeholder. |

## Coordinates

- `com.oracle.spring.ai:spring-ai-oracle-parent:2.0.0-M6`
