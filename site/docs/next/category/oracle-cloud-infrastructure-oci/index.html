Oracle Cloud Infrastructure (OCI)

Authentication, configuration, and Spring Boot integrations for OCI services.

📄️ OCI Core

Spring Cloud Oracle provides a core starter that auto-configures OCI authentication and shared infrastructure for the service-specific starters.

📄️ Configuration Properties

This page summarizes the common Spring Cloud Oracle OCI properties and the subset that supports configuration refresh at runtime.

📄️ Autonomous Database

OCI Autonomous Database is a managed data service that automates patching, upgrades, tuning, and routine maintenance.

📄️ Object Storage

OCI Object Storage stores arbitrary files and objects. Spring Cloud Oracle adds both a storage API and a Spring Resource integration.

📄️ Vault

OCI Vault can be used both as a Spring property source and as an application bean for managing secrets.

📄️ Streaming

OCI Streaming is a managed service for ingesting and consuming high-volume data streams in real time.

📄️ Queues

OCI Queue is a managed queueing service for independently processed messages without loss or duplication.

📄️ Notifications

OCI Notifications is a pub/sub service for delivering alerts and messages to email, SMS, Functions, Slack, PagerDuty, and custom HTTPS endpoints.

📄️ Logging

OCI Logging is a managed service for collecting and searching logs from OCI resources and applications.

📄️ Email Delivery

OCI Email Delivery provides Spring Mail integrations through MailSender and JavaMailSender.

📄️ Functions

OCI Functions is a fully managed functions-as-a-service platform. Spring Cloud Oracle provides a client for invoking deployed functions.

📄️ Oracle NoSQL Database

Oracle NoSQL Database provides managed JSON, table, and key-value storage with flexible transaction guarantees.

📄️ Generative AI

OCI Generative AI provides managed chat and embedding models through Spring-friendly abstractions.
